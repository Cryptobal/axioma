## Por qué integrar
Sin integraciones, el ERP se convierte en un silo. Con APIs modernas, habilitas automatización, reportes confiables y experiencias digitales coherentes.

## Patrones
- API Gateway y autenticación
- Webhooks y colas para eventos
- ETL controlado para legados
- Versionado y compatibilidad

## Buenas prácticas
Trazabilidad, observabilidad, límites de tasa, pruebas contractuales y documentación viva.

## Primeros pasos
Define un dominio acotado (por ejemplo, órdenes), diseña contratos y habilita un piloto con métricas.
